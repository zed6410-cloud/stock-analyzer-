# StockAI 주식 분석기 - 설치 및 실행 방법

## 1단계: Node.js 설치
https://nodejs.org 에서 LTS 버전 다운로드 후 설치

## 2단계: 백엔드 패키지 설치
```
cd backend
npm install
```

## 3단계: Claude API 키 설정 (선택 - AI 분석 기능)
1. https://console.anthropic.com 에서 API 키 발급
2. `backend/.env.example` 파일을 복사하여 `backend/.env` 로 저장
3. `ANTHROPIC_API_KEY=sk-ant-...` 에 본인 키 입력

## 4단계: 프론트엔드 패키지 설치
```
cd frontend
npm install
```

## 5단계: 실행
`start.bat` 더블클릭 또는:
```
# 터미널1 (백엔드)
cd backend && npm start

# 터미널2 (프론트엔드)
cd frontend && npm start
```

## 주요 기능
- 한국/미국 주식 검색 (이름 또는 티커)
- 실시간 현재가, 시가, 종가, 거래량
- 1개월~5년 가격 차트
- 손익계산서, 재무상태표, 현금흐름표
- 핵심 재무 지표 (ROE, ROA, PER, PBR 등)
- 애널리스트 컨센서스 목표주가
- AI 기반 목표가/하향가 분석 (Claude API 필요)

## 한국 주식 검색 방법
- 삼성전자: `005930.KS`
- 카카오: `035720.KS`
- 네이버: `035420.KS`
- 또는 검색창에 "삼성" 이라고 입력하면 자동완성
